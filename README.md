# Flutter Calculator App

A simple and clean Calculator App built using Flutter and Dart.

This project demonstrates the basic concepts of Flutter development such as:

* StatefulWidget
* TextEditingController
* User input handling
* Arithmetic operations (+, -, *, /)
* UI design using TextField, ElevatedButton, Column, Wrap, and Stack

## Features

✔ Addition Operation
✔ Subtraction Operation
✔ Multiplication Operation
✔ Division Operation
✔ Clean and modern UI
✔ Responsive design
✔ Beginner-friendly Flutter project

## Technologies Used

* Flutter
* Dart
* Material Design

## Output Screen

The app contains:

* AppBar with title
* First number input field
* Second number input field
* Operation buttons (+, -, *, /)
* Result display section
* Custom footer text
